## Root Cause Summary

The play directory was named `garura-capture` instead of `capture`. In Meridian, the directory name under `core/components/plays/` determines the slash-command invocation after `/sync-claude` deploys to `~/.claude/skills/<dirname>/`. The `name: "garura:capture"` frontmatter (namespace identifier) was correct but the directory name was not — introduced during the #266 rename using the full namespaced form instead of the simple invocation name.

## Fix Strategy Applied

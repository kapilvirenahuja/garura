# Workflow Selection — define rebake (issue #370)

Current structure: **A — full checkpoint flow** (Pre-flight → Preparation →
Checkpoint → Execution → Evidence). 11 phases, two human Tether/Orbit/Vanish
gates (Phase 5 placement, Phase 9 epic).

Assessment after C17 + F11 addition:
- C17 (Decision Surfacing Discipline) and F11 add a play-owned surfacing
  step immediately before each existing checkpoint gate (before Step 5a and
  before Step 9a). This is additive within the checkpoint phases.
- It does not introduce a new agent, remove a phase, or change the play's
  complexity class. Decision surfacing at human gates is precisely what
  Structure A's checkpoint phase exists for.

Decision: **keep Structure A. No structure switch.**

# discover-product

Given a product idea or problem statement, discover the market opportunity, draft a product vision with strategic goals and market positioning, validate the vision for completeness, and lock it when ready. The recipe runs as a single invocation with inline checkpoints. It produces a vision artifact (product.yaml), an interactive HTML brief (product-brief.html) for human review, validates completeness, and locks the vision — all in one flow. All product artifacts are stored in project-scoped product STM. Supports both market-facing products and internal libraries.

## Workflow

12 steps, single flow. One invocation: assess → discover → draft → derive-profiles → brief → checkpoint → validate → checkpoint → lock → evidence → summary.

### Step 1 — Assess Intent

Owner: recipe (no agent call)
Depends on: pre-flight

Before any agent calls, assess the intent to determine product type (C13) and whether opportunity discovery should be skipped (C12).

**Three-signal assessment (C12):**

Evaluate the intent against three signals:
1. The intent names specific consumers rather than market segments
2. The intent describes a technical interface boundary rather than a user problem
3. No market-facing end users are identifiable

```
product_type = "product"    # default
skip_opportunity = false

if (intent names specific consumers, NOT market segments)
   AND (intent describes technical interface boundary, NOT user problem)
   AND (no market-facing end users identifiable):
  product_type = "library"
  skip_opportunity = true

# C13: Record product_type for product.yaml type field
# C12: Record skip_opportunity for Step 2 conditional
```

**When `skip_opportunity = true`:**
- Skip Step 2 entirely
- Pass `raw_intent` directly to Step 3 (draft-vision) instead of `market_context`
- C5 domain agent budget: 2 base (draft + validate) plus up to 2 cycle-back = 4 max

**When `skip_opportunity = false`:**
- Proceed to Step 2 normally
- C5 domain agent budget: 3 base (discover + draft + validate) plus up to 1 cycle-back = 4 max

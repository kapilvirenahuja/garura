# briefs

Scan a product directory for YAML design artifacts, checksum each one against stored values, regenerate the corresponding HTML brief for any that changed, and rebuild hub.html. Context-aware — when run from an epic branch, scope regeneration to that epic's artifacts plus product-level artifacts; when on main or with no epic context, scan product-level artifacts and all epics with YAMLs on disk. No flags, no arguments — derives everything from the execution context.

### Phase: Checksum Comparison

**Step 2 — Compute and Compare Checksums**
Owner: play
Depends on: Step 1

1. Read `{product_base}/briefs/.checksums.json` — create empty `{}` if missing (C6)
2. For each YAML in scope:
   a. Compute sha256 of the file content
   b. Look up the relative key in .checksums.json
   c. If checksum differs or key is missing → add to `changed` list
   d. If checksum matches → skip (C2)
3. Build the `changed` list with artifact names grouped by scope:
   - `product_changed`: list of product-level artifact names (e.g., `["product", "roadmap"]`)
   - `epic_changed`: map of `{epic_id: [artifact_names]}` (e.g., `{"E1": ["features", "architecture"]}`)
4. If `changed` is empty → report "All briefs up to date." with hub path, exit (S4)

**Step 2 Evals:**

- **SE-10 (C2):** Staleness is determined solely by comparing a cryptographic checksum of each YAML artifact against the stored checksum from the previous run. A brief is regenerated if and only if its source YAML checksum differs from the stored value or no stored value exists. Pass: The regeneration list exactly matches the set of YAMLs whose sha256 differs from .checksums.json. No other criteria (mtime, file size) influence the decision. Fail: Regeneration decision uses criteria other than checksum comparison, or a YAML with changed checksum is not regenerated, or a YAML with unchanged checksum is regenerated.

- **SE-2 (F2):** No YAML artifact whose checksum matched the stored value had its brief regenerated. Pass: The changed list contains only artifacts whose checksums differed from stored values. Unchanged artifacts are absent from the changed list. Fail: A YAML artifact whose checksum has NOT changed appears in the changed list.

- **SE-7 (F7):** On an epic branch, only the scoped epic's artifacts and product-level artifacts are in the regeneration scope. Pass: The changed list contains only artifacts from the resolved epic and product-level artifacts. No artifacts from other epics appear. Fail: Artifacts from unrelated epics are regenerated when only the scoped epic's artifacts and product-level artifacts should be in scope.

- **SE-13 (C7):** When epic context is resolved, product-level artifacts are always included in the scan alongside the scoped epic's artifacts. Pass: The scan scope includes product.yaml and roadmap.yaml (if they exist) regardless of which epic is in context. Fail: Product-level artifacts are excluded from the scan when epic context is resolved.

### Phase: Execution

**Step 3 — Generate Changed Briefs**
Owner: play (direct, no agent delegation)
Depends on: Step 2 (changed list non-empty)

For each YAML in the `changed` list, perform the following operations directly:

**3.1 — Convert YAML to JSON**

```bash
python3 -c "
import sys, yaml, json
with open(sys.argv[1]) as f:
    data = yaml.safe_load(f)
json_path = sys.argv[1].rsplit('.yaml', 1)[0] + '.json'
with open(json_path, 'w') as f:
    json.dump(data, f, indent=2, default=str)
" "{yaml_path}"
```

Run this for each changed YAML. The JSON file is written alongside the YAML (same directory, same base name, `.json` extension). JSON is an isomorphic representation — no transformation, every top-level YAML key maps to a top-level JSON key (C11).

**3.7 — Generate hub.json manifest**

Run after all per-artifact copies complete. This regenerates the full hub manifest regardless of which artifacts changed.

```bash
python3 -c "
import sys, os, json, yaml, glob

product_base = sys.argv[1]
slug = sys.argv[2]
artifact_base = product_base
briefs_dir = os.path.join(artifact_base, 'briefs')

hub = {
    'product_name': slug,
    'slug': slug,
    'generated_at': __import__('datetime').datetime.now().isoformat(),
    'product_level': [],
    'epics': []
}

# Product-level artifacts
artifact_paths = {'product': os.path.join(artifact_base, 'discovery', 'product.yaml'),
                  'roadmap': os.path.join(artifact_base, 'roadmap', 'roadmap.yaml')}
for artifact in ['product', 'roadmap']:
    yaml_path = artifact_paths[artifact]
    if os.path.exists(yaml_path):
        with open(yaml_path) as f:
            data = yaml.safe_load(f) or {}
        stat_map = {'product': ('Strategic Goals', len(data.get('strategic_goals', []))),
                     'roadmap': ('Epics', len(data.get('timeline', [])))}
        stat_label, stat_value = stat_map.get(artifact, ('Items', 0))
        brief_path = artifact + '-brief.html'
        hub['product_level'].append({
            'artifact': artifact,
            'status': data.get('status', 'DRAFT'),
            'has_brief': os.path.exists(os.path.join(briefs_dir, brief_path)),
            'brief_path': brief_path,
            'stat_label': stat_label,
            'stat_value': stat_value
        })
        hub['product_name'] = data.get('slug', slug)

# Epic artifacts
epics_dir = os.path.join(artifact_base, 'roadmap', 'epics')
if os.path.isdir(epics_dir):
    for epic_id in sorted(os.listdir(epics_dir)):
        epic_dir = os.path.join(epics_dir, epic_id)
        if not os.path.isdir(epic_dir):
            continue
        epic_entry = {'epic_id': epic_id, 'artifacts': []}
        stat_map = {'features': 'Features', 'architecture': 'Principles', 'tech': 'Components',
                     'scenarios': 'Scenarios', 'plan': 'Features'}
        for artifact in ['features', 'architecture', 'tech', 'scenarios', 'plan']:
            yaml_path = os.path.join(epic_dir, artifact + '.yaml')
            if os.path.exists(yaml_path):
                with open(yaml_path) as f:
                    data = yaml.safe_load(f) or {}
                # Count stat
                count_keys = {'features': 'features', 'architecture': 'principles', 'tech': 'components',
                              'scenarios': 'groups', 'plan': 'execution_order'}
                count = len(data.get(count_keys.get(artifact, ''), []))
                brief_path = 'epics/' + epic_id + '/' + artifact + '-brief.html'
                epic_entry['artifacts'].append({
                    'artifact': artifact, 'status': data.get('status', 'DRAFT'),
                    'has_brief': os.path.exists(os.path.join(briefs_dir, brief_path)),
                    'brief_path': brief_path,
                    'stat_label': stat_map.get(artifact, 'Items'),
                    'stat_value': count
                })
        if epic_entry['artifacts']:
            hub['epics'].append(epic_entry)

with open(os.path.join(briefs_dir, 'hub.json'), 'w') as f:
    json.dump(hub, f, indent=2)
" "{product_base}" "{slug}"
```

# algorithm

Optional play that reads tech.yaml, detects algorithmically complex `internal_interfaces` entries (via 11 indicator patterns or an explicit `algorithm_spec_required: true` flag), dispatches `tech-architect` to produce `reference-algorithms.md`, and gates the result on human approval before locking. CRUD epics with no qualifying interfaces exit with a skip notice — no agent dispatch occurs.

# capture-learning

Given a completed epic, reconcile the full product LTM by comparing what prepare-epic knew at planning time against what actually happened during implementation and validation. Three-tier analysis: Tier 1 foundational artifacts are check-only (ADR if changed), Tier 2 artifacts are enriched with implementation outcomes, Tier 3 additions are proposed for new content. After reconciliation and human approval, archive the issue's STM directory. The issue must be in a closed state before any work proceeds.

```json

      "epic_id": "{epic_id}"
    },
    "output": {
      "proposals_path": "{stm_base}/{issue}/evidence/capture-learning/reconciliation-proposals.yaml"
    }
  }
}
```

Agent reads context baseline (what prepare-epic knew), reads implementation outcomes (milestones, verdicts, status-reports, e2e-results), consumes check-drift output if present, compares against current product LTM, and produces tiered reconciliation proposals.

**Step 3 Evals:**

- **SE-2 (C6):** Every proposal is classified into exactly one tier (1, 2, or 3). No unclassified findings.
- **SE-3 (C7):** Proposals reference specific evidence from context/ or evidence/ — not generic summaries.
- **SE-4 (C8/F11):** If drift_manifest_path was provided and file exists, proposals with `source: check-drift` are present. No drift finding is re-derived from scratch when the manifest covers it.
- **SE-5 (C12/F8):** Every Tier 1 proposal includes an `adr_proposal` block AND an `impact_assessment` block. Missing either is a structural failure.
- **SE-6 (C13):** If any product LTM tier's artifacts are absent, reconciliation-proposals.yaml records a `tiers_skipped` entry with the tier and reason — not a hard failure.
- **SE-7 (C14):** If a Tier 2 proposal targets an epic's post_implementation, the proposed content matches the intent-epic.yaml post_implementation schema fields.
- **SE-8 (C15):** When total_proposals is 0, proposals list is empty, and summary reflects zero findings across all tiers.

**Zero-proposal case (C15):** If reconciliation produces no proposals, the play skips Steps 4-5 and proceeds to archival (Step 6).

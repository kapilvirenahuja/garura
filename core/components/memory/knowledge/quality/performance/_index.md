# Performance Quality Standards

Performance quality covers the practices that make systems fast, measurable, and reliable under load. These files provide the assessment checklist items (PERF-01 through PERF-24) used by the quality-check skill.

## Files

- [load-testing-profiling.md](load-testing-profiling.md) — Load test tooling, profiling practices, benchmarks, and CI integration | Patterns: k6, artillery, jmeter, locust, flame graph, benchmark, performance budget
- [slos-reliability.md](slos-reliability.md) — SLO/SLI definitions, error budgets, availability targets, incident classification, MTTR/MTTD | Patterns: SLO, SLI, error budget, MTTR, MTTD, incident, reliability review
- [resource-optimization.md](resource-optimization.md) — Memory leaks, connection pooling, caching, CDN, lazy loading, N+1 queries, bundle size | Patterns: memory leak, connection pool, caching, CDN, lazy loading, bundle size, N+1, index
